# reference/ — 1단계 산출물

사용자가 레퍼런스를 제공하면 여기에 둔다(또는 1단계에서 수집):

- `source.png` — 레퍼런스 스크린샷
- `dom.html` — (옵션) 캡처한 DOM
- `signals.json` — (옵션) 레퍼런스에서 뽑은 관찰 신호

**주의:** 루프의 비교 기준은 여기(`reference/`)가 아니라 `design/` 이다(원칙 1-2).
이 폴더는 2단계 디자인 도출의 입력일 뿐, 채점의 정답지가 아니다.
(픽셀 신호만 예외적으로 `reference/components/<id>.png` 크롭을 쓸 수 있으나 기본 비활성.)

이 예제 run 은 레퍼런스 없이 디자인을 바로 선언했으므로 이 폴더는 비어 있다.
