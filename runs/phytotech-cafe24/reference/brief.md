# 디자인 브리프 — phytotech-cafe24 (1단계 → 2단계 입력)

레퍼런스 `autowash.co.kr` 의 UI/UX 룩앤필을 **건강기능식품 브랜드**로 재해석한다.
베끼기 아님(원칙 1-2). 아래는 2단계(design/ 도출)에서 확정 반영할 사용자 결정.

## 확정 결정
- **업종/브랜드**: 건강기능식품 (레퍼런스 업종=자동차 워셔액과 무관, 룩앤필만 차용)
- **납품 범위**: 풀 페이지 (export 조각도 풀 페이지)
- **폰트**: Pretendard 유지 (레퍼런스와 동일 패밀리)
- **색 — 단일 액센트 구조 유지, 색조만 변경**:
  - 액센트: 블루(#0099FF) → **그린/내추럴 계열** (건강·신뢰 톤)
  - 다크 앵커(히어로/푸터 ~#1B1E27): **딥그린/차콜**로 재해석 가능
  - 베이스: 화이트 + 옅은 회색(#F8F8F8) 섹션 구분 유지
  - 텍스트: primary #111, muted #757575 유지
- **셰이프**: pill/라운드 코너, 넉넉한 여백 vs 고밀도 그리드 대비 유지

## 컴포넌트 (components.json 배열 순서 = 페이지 흐름)
1. `header` — sticky, 로고 좌 + pill 검색바 + 가로 GNB + 우측 아이콘/로그인
2. `hero` — 캐러셀(다크 배경, 굵은 카피 좌 + 제품 우)
3. `category` — 원형 카테고리 퀵메뉴(둥근 썸네일 가로 행)
4. `product-grid` — 4~5열 상품 카드(이미지/타이틀/가격/배지) + 섹션 헤더
5. `footer` — 다크 배경, 회사정보/약관/고객센터, 고밀도·저위계

## 재해석 메모
- 히어로 톤: 공구의 강한/메탈 → 깨끗·신뢰 헬스 비주얼(성분/효능 카피)
- 상품 카드 배지: 성능/증정 → 효능·성분·정기구독 신뢰 메시지
- 인터랙션(6단계): 헤더 sticky, 히어로 자동 슬라이드, 맨위로 버튼

## 미정(2단계에서 확인/가정)
- 브랜드 컬러/로고 자산: 미제공 → 그린 계열로 도출(나중에 교체 가능)
