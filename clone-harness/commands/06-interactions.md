6단계: 프론트엔드 인터랙션. **루프 밖이고 유사도 채점 대상이 아니다** — 기능의 기준은 "유사"가 아니라 "동작".

- `runs/$ARGUMENTS/src/interactions.js` 에 구현. 스크롤 동작, 토글, 메뉴, 폼 검증 등 프론트 한정.
- 백엔드/DB/비공개 API 는 범위 밖(외부에서 복제 불가).
- 대화로 요구를 받아 구현하고, 동작을 직접 확인한다. 완료가 두 번째 사람 게이트.

납품까지 가려면: `/07-export <run-name>`.
